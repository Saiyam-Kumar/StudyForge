# 📚 StudyForge

StudyForge is a desktop productivity application built with **Java, Swing, JDBC, and MySQL** that helps students manage subjects, assignments, deadlines, study sessions, and academic progress in one place.

The goal of this project is to create a practical tool that improves study planning while applying core Java concepts learned throughout my Java development journey.

---

## ✨ Features

### 📖 Subject Management
- Add, update, and delete subjects
- Store subject information and exam dates
- Organize academic workload efficiently

### ✅ Task Management
- Create daily study tasks
- Mark tasks as completed
- Track pending and overdue tasks

### 📅 Deadline Tracking
- Assignment deadlines
- Exam reminders
- Upcoming academic events

### ⏱️ Pomodoro Timer
- 25-minute focus sessions
- 5-minute break sessions
- Built using Java Multithreading

### 🔥 Study Streaks
- Track daily study consistency
- Maintain learning habits
- Visual progress tracking

### 📊 Dashboard
- Total tasks completed
- Pending tasks
- Study progress overview
- Productivity statistics

### 💾 Persistent Data Storage
- MySQL database integration
- Data remains available after application restart
- JDBC-based database connectivity

### 📂 Export Functionality
- Export study plans
- Save reports to text files
- File Handling and I/O Streams implementation
